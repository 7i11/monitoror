## Introduction

Monitoror is a unified monitoring wallboard to check services, servers, builds, ...

Supported platforms:

- Linux (64bits, ARM)
- macOS
- Windows (64bits)

[Visit the Monitoror website for more details](https://monitoror.github.io/monitoror/)
